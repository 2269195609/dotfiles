安装
----

Install::

    # install vundle
    git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
    # install vim-powerline
    git clone --branch=develop https://github.com/Lokaltog/vim-powerline ~/.vim/bundle/
    cp .vimrc ~
    cp .gvimrc ~
    # vim :BundleInstall
